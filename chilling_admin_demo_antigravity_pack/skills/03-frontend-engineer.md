# Skill - Frontend Engineer for Antigravity

## Stack

```txt
Vite
Vue 3
TypeScript
Tailwind CSS
Pinia
Vue Router
Lucide icons
ECharts
```

## Quy tắc code

```txt
1. Component nhỏ, không viết file quá dài.
2. Data mock để trong src/data.
3. Type để trong src/types.
4. Layout để trong src/components/layout.
5. UI reusable để trong src/components/ui.
6. Brand component để trong src/components/brand.
7. Page để trong src/pages.
8. Không hard-code quá nhiều trong template.
9. Format tiền bằng utility riêng.
10. Không dùng any nếu tránh được.
11. Responsive mobile-first.
```
